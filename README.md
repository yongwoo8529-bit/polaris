# Polaris - 당신의 내면의 길을 찾아서

**Polaris(북극성)**는 청소년들이 공부라는 틀에 매료되지 않고, 자신만의 진짜 길을 찾을 수 있도록 돕는 심리 상담 및 성향 분석 앱입니다.

## 🌌 주요 특징

### 1. 프리미엄 천문 디자인
- **Deep Space 테마**: 어두운 우주 배경과 세련된 그라데이션, 유리 질감(Glassmorphism)을 사용한 UI.
- **Orbitron 타이포그래피**: 미래지향적이고 전문적인 느낌의 폰트 적용.

### 2. 5가지 은하계 설문 (총 20문항)
사용자의 답변(1~5점 가중치)에 따라 각 파트마다 최대 20점의 점수가 부여됩니다.
- **보라색 (탐구의 은하)**: 호기심과 정보 탐구력.
- **청색 (질서의 은하)**: 논리와 체계적인 정리 능력.
- **주황색 (온기의 은하)**: 공감과 사람 간의 연결.
- **금색 (울림의 은하)**: 자기 표현과 영향력.
- **녹색 (울타리의 은하)**: 협력과 책임감.

### 3. 극적인 결과 연출 (Three.js 기반)
- **북극성 형성**: 설문이 끝나면 화면이 어두워지며 "당신의 내면의 북극성이 만들어집니다"라는 문구와 함께 가장 높은 점수를 얻은 파트의 색상으로 중심 북극성이 나타납니다.
- **인터랙티브 은하 폭발**: 중심 북극성을 클릭하면, 사용자의 가중치에 따라 **5개의 나선 은하**가 생성되어 팽창합니다.
- **가중치 반영**: 각 은하의 크기와 파티클 밀도는 사용자가 해당 파트에서 얻은 점수에 비례하여 동적으로 생성됩니다.

## 🛠 실행 방법

1. `index.html`, `style.css`, `app.js` 파일을 동일한 폴더에 저장합니다.
2. 브라우저에서 `index.html` 파일을 엽니다.
3. "탐험 시작하기" 버튼을 눌러 설문을 진행하세요.

## 🔬 기술 스택
- **Three.js**: 고해상도 3D 파티클 시스템 렌더링.
- **GSAP**: 부드러운 UI 전환 및 애니메이션.
- **Vanilla CSS3**: 글래스모피즘 및 동적 레이아웃.
- **Canvas API**: 배경의 미세한 별빛 효과.
