서비스 개요

POLARIS는 홀랜드 직업적성이론을 기반으로 한 48문항 진로성향 검사 도구입니다. 학생이 검사를 완료하면 6가지 성향 점수가 산출되고, 결과가 3D 은하계로 시각화됩니다. 교사는 학급 코드를 발급해 학생 결과를 실시간으로 확인하고 PDF·Excel로 내보낼 수 있습니다.

Step 1 · 교사 대시보드 접속
1
대시보드 URL 접속
아래 주소로 접속하거나, 랜딩 페이지 하단 교사 입장 → 링크를 클릭합니다.
https://polaris-umber.vercel.app/teacher.html
2
비밀번호 입력
접속 시마다 비밀번호를 입력합니다.
PW polaris2026
비밀번호는 외부에 공유하지 마세요.
Step 2 · 학급 코드 발급
1
학급 코드 발급 탭 선택
대시보드 상단 학급 코드 발급 탭을 클릭합니다.
2
정보 입력 후 생성
학교명 · 선생님 성함 · 학급명을 입력하고 학급 코드 생성하기를 클릭합니다.
세 항목이 모두 일치하는 학급이 이미 있으면 기존 코드를 불러옵니다.
같은 학교·교사·학급으로 재접속하면 항상 동일한 코드가 나옵니다.
3
QR 코드 인쇄
생성된 코드 하단 QR 인쇄 버튼을 눌러 A4 용지에 출력하거나 화면에 띄워 학생들이 스캔하도록 합니다.
Step 3 · 학생 참여 방법
📱 QR 코드 스캔 (권장)
  • 스마트폰 카메라로 QR 스캔
  • 학급 코드 자동 입력됨
  • 이름 입력 후 개인정보 동의
  • 검사 시작
⌨️ 코드 직접 입력
  • 학생이 polaris.html 직접 접속
  • 학급 코드 6자리 수동 입력
  • 이름 입력 후 개인정보 동의
  • 검사 시작
검사 소요 시간: 약 10~15분 / 총 48문항 (6파트 × 8문항)
Step 4 · 결과 실시간 확인
1
결과 조회 탭 선택
대시보드 상단 결과 조회 탭 → 학급 코드 6자리 입력 → 조회하기 클릭
2
실시간 갱신 확인
결과 목록은 10초마다 자동 갱신됩니다. 수업 중 학생들이 제출하면 즉시 반영됩니다.
3
개별 PDF 저장
각 학생 행 오른쪽 [PDF] 버튼 → 새 탭에서 보고서 열림 → 인쇄 / PDF 저장
4
전체 Excel 내보내기
목록 상단 [📊 Excel 내보내기] 클릭 → POLARIS_학급코드_결과.csv 파일 다운로드
엑셀에서 열면 이름·성향유형·6파트 점수·날짜가 열 형태로 정리됩니다.
5
학생 데이터 삭제
각 행 오른쪽 [삭제] 버튼으로 오입력된 데이터를 제거할 수 있습니다.
자주 묻는 질문
학생이 실수로 이름을 잘못 입력하고 제출했어요.
결과 조회 화면에서 해당 행의 [삭제] 버튼으로 제거한 뒤, 학생에게 다시 검사하도록 안내하세요.
같은 학생이 두 번 제출하면 어떻게 되나요?
같은 학급 코드 + 이름 조합으로 재제출 시 이전 결과를 덮어씁니다. 중복 데이터는 생기지 않습니다.
학급 코드를 잊어버렸어요.
동일한 학교명·선생님 성함·학급명을 입력하면 기존 코드를 다시 불러옵니다.
모바일에서 PDF 저장이 안 돼요.
iOS는 Safari 브라우저 → 공유 → 파일에 저장, Android는 Chrome → 인쇄 → PDF로 저장을 선택하세요.
학생 검사는 어떤 기기로 해야 하나요?
스마트폰, 태블릿, PC 모두 가능합니다. QR 스캔은 스마트폰 카메라 앱을 사용하세요.
