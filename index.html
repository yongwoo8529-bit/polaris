Features

기존 진로검사와 다릅니다

학생 스스로 탐색하고, 교사가 수업에 연결하는 진로교육 도구입니다.
텍스트 결과지 대신 인터랙티브 3D 은하계로 성향을 시각화합니다.

🌌

3D 진로 은하 시각화

검사 완료 후 Three.js 3D 인터랙션으로 자신의 성향이 은하계로 펼쳐집니다. 결과 화면은 학생마다 다르게 나타나 자연스러운 진로 대화의 출발점이 됩니다.

🔭

홀랜드 6유형 기반 자기탐색 도구

Holland Career Code(R·I·A·S·E·C) 직업적성이론의 6가지 유형 체계를 기반으로 한 자기탐색 도구입니다. 직업 정보는 한국직업능력연구원 커리어넷과 공식 연계합니다.

📊

교사 관리 대시보드

학급 코드를 발급하면 학생 결과가 자동 집계됩니다. 6가지 성향 유형 분포와 학생별 결과를 한눈에 확인할 수 있습니다.

📄

결과 PDF 보고서

학생 · 학부모 상담용 인쇄 보고서 자동 생성. 진로수업 기록 자료나 가정통신문 첨부 자료로 활용할 수 있습니다.

🏫

학교 단위 계정

학교 코드로 교사 등록 → 학급 생성 → 초대 코드로 학생 참여. 별도 앱 설치 없이 브라우저에서 바로 사용합니다.
